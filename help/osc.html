OSC Command Reference

as of: 06-Nov-15
warning: OSC command set may be subject to changes.

  • /dumpOSC <(int) incomingMode> [<(int) outgoingMode>]
  • /main
    • quit [<(int) force=(0|1)>]
    • query properties:
      version : <(String) version>
  • /doc
    • open <(String) path> [<(int) visible=(0|1)>]
      replies to sender with either /done /doc open <path> <(Int) id> or /failed /doc open <path>
    • new <(String) processName>
    • query properties:
      count : <(int) numberOfDocs>
  • /doc/<address>
    • where <address> is one of:
      • active
      • index/<(int) idx>
      • id/<(int) id>
    • close [<(int) force=(0|1)>]
    • start
    • stop
    • pause
    • resume
    • activate
    • query properties:
      id : <(int) id>
      dirty : <(int) isDirty=(0|1)>
      name : <(String) name>
      file : <(String) documentFilePath>
      process : <(String) processName>
      running : <(int) isRunning=(0|1)>
      progression : <(float) progression=(0.0...1.0)>
      error : <(String) errorMessage>

querying single values

query <id> [ <(String) property> ] * N

replies with

/query.reply <id> [ <(Object) value> ] * N

getting complex objects

get <id> <... args>

replies with

/get.reply <id> <... args>
